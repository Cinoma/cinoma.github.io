Chris Climbs Code - Portfolio ​

A personal portfolio and blog built with VitePress, showcasing projects, technical guides, and career journey updates.

🚀 Features ​

  • Responsive design
  • Project showcase
  • Technical blog
  • GitHub/GitLab activity integration
  • Career journey updates

🛠️ Tech Stack ​

  • VitePress - Static Site Generator
  • Vue.js - Frontend Framework
  • GitHub Pages - Hosting

🏃‍♂️ Getting Started ​

Prerequisites ​

  • Node.js (version 16 or higher)
  • npm or yarn

Installation ​

  1. Clone the repository
bash
git clone https://github.com/Cinoma/vite-press-portfolio.git
cd vite-press-portfolio
  1. Install dependencies
bash
npm install
  1. Start development server
bash
npm run docs:dev
  1. Build for production
bash
npm run docs:build
  1. Preview production build
bash
npm run docs:preview

📁 Project Structure ​

.
├── .vitepress/
│   └── config.mts     # VitePress configuration
├── public/            # Static assets
├── projects/          # Project documentation
├── blog/             # Blog posts
└── about.md          # About page

🚀 Deployment ​

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.

Visit the live site: https://cinoma.github.io/vite-press-portfolio/
